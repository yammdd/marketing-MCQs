# marketing-MCQs

A simple web-based quiz system for final exam revision. It helps users practice multiple-choice questions and receive instant feedback. This tool is for review purposes only and does not guarantee exam coverage.

## Run locally with Docker
### 1. Download
```bash
git clone https://github.com/yammdd/marketing-MCQs.git
cd marketing-MCQs
```

### 2. Run
```bash
docker-compose up -d --build
```

Then open:

```
http://localhost:8080
```
